# Spam-Mail-Classifier
📌 Project Overview

This project classifies emails as spam or ham using NLP and machine learning.

🛠️ Technologies Used

🐍 Python

🧠 Scikit-learn

📚 NLTK


📂 Dataset

The dataset is sourced from Kaggle.

Contains a collection of emails labeled as spam or not spam.
